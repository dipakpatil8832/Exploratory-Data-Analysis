# Customer Churn Analysis (EDA Project)

## 1. Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a telecom customer dataset to identify patterns and factors that influence **customer churn**. The analysis helps understand why customers leave a service and which segments are more likely to churn.

## 2. Objectives

* Analyze customer demographics and service usage.
* Identify factors influencing customer churn.
* Explore relationships between features using statistical and visual analysis.
* Provide insights that can help improve customer retention strategies.

## 3. Dataset Information

The dataset contains telecom customer information including:

* **customerID** – Unique ID of each customer
* **gender** – Male or Female
* **SeniorCitizen** – Whether the customer is a senior citizen (1 = Yes, 0 = No)
* **Partner** – Whether the customer has a partner
* **Dependents** – Whether the customer has dependents
* **tenure** – Number of months the customer has stayed with the company
* **PhoneService / InternetService** – Type of service used
* **Contract** – Contract type of the customer
* **PaymentMethod** – Payment method used
* **MonthlyCharges** – Monthly amount charged
* **TotalCharges** – Total amount charged
* **Churn** – Whether the customer left the service (Yes/No)

## 4. Tools & Technologies

* **Python**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Seaborn**
* **Jupyter Notebook**

## 5. Analysis Performed

1. Data Cleaning and preprocessing
2. Handling missing values
3. Converting categorical variables
4. Exploratory Data Analysis (EDA)
5. Univariate, Bivariate, and Multivariate analysis
6. Data visualization to identify churn patterns

## 6. Key Insights

* Customers with **month-to-month contracts** have a higher churn rate.
* **Higher monthly charges** are associated with increased churn.
* Customers with **longer tenure** are less likely to churn.
* Certain **payment methods and services** influence churn behavior.
